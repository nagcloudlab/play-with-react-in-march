
why do we need javascript?

-> to develop dynamic web-UI. ( e,g browser )
-> to develop server-side applications. ( e,g Node.js )
-> to develop mobile applications ( e,g React Native )
-> to develop desktop applications ( e,g Electron )


javascript features:

-> single-threaded-process
-> by default, all I/O operations are asynchronous ( non-blocking )
-> function is a first-class citizen ( can be treated as a value )
-> dynamically-typed language ( no need to declare variable type )


How to run javascript code?

-> using javascript engine.

1. chrome -> v8 ( c++ )
2. firefox -> spider monkey ( c++ )
3. safari -> JavaScriptCore ( c++ )
4. Edge -> Chakra ( c++ )
5. Node.js -> v8 ( c++ )
6. JVM -> Nashorn ( java )
7. MongoDB -> v8 ( c++ )
....    

javasctipt follows ECMAScript standard. ( e,g ES6, ES7, ES8, ... )
